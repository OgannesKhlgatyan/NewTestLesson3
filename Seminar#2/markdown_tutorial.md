# Туториал по языку разметки MarkDown

## Списки 

Для разметки неупорядоченных списков можно использовать
```
 или * , или - , или + :
```
* С помощью " * "
- С помощью " - "
+ С помощью " + "

Вложенные пункты создаются четырьмя пробелами перед маркером
пункта:
* Пункт 1
    * Пункт 1.1
        * Пункт 1.1.1
            * И так далее
